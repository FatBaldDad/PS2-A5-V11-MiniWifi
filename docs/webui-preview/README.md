# WebUI Preview

This directory contains a polished static mockup of the PS2-A5-V11-MiniWifi WebUI.

## Quick Preview (Windows)

```bat
cd docs\webui-preview
py -m http.server 8080
```

Then open:

`http://localhost:8080`

## Notes

- This is a static HTML/CSS/JS mockup.
- It uses mock data only.
- No React, Node, npm, Bootstrap, or build tools are required.
